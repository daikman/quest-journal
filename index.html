How to use the Quest Journal

Change the names of quests and tasks by clicking on them and directly editing the text.

Add new quests and tasks using the NEW QUEST or ADD TASK button, respectively.

Add sub-tasks using the "+" button next to a task.

Remove any item using the adjacent "-" button.

Currently, the SAVE button downloads your quests as a text file. Then the LOAD button asks for you to upload a previously saved text file.
